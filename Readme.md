# FriTest
## Descripción
Obtener y listar 20 elementos diferentes desde la [API](https://www.themealdb.com/api/json/v1/1/random.php).

## Requerimientos para correr aplicación
1. Xcode 15.0.1 o superior.
2. iOS 17 o superior.
